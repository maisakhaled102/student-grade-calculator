# 🎓 نظام حساب المعدلات الذكي | Smart GPA Calculator

تطبيق احترافي مطور بواجهة مستخدم رسومية (GUI) حديثة وجذابة، مصمم بلغة **Python** وباعتماد مكتبة **Tkinter** لمساعدة الطلاب في حساب أداؤهم الأكاديمي بدقة وسهولة.

---

## 🚀 المميزات الرئيسية (Key Features)

*   **📊 دعم 5 مواد دراسية:** استقبال وفحص درجات 5 مواد مختلفة في نفس الوقت.
*   **🎯 حساب الـ GPA التراكمي:** حساب تلقائي دقيق للمعدل التراكمي من (**4.0**).
*   **🎨 تصميم عصري (Dark Theme):** واجهة مستخدم ملوّنة ومريحة للعين بخلفية داكنة متناسقة.
*   **🛡️ منظومة فحص المدخلات:** حماية البرنامج من الانهيار والتحقق من أن الدرجات تقع بين (0-100).
*   **🌐 ربط مباشر بالمنصة:** زر مدمج ينقل المستخدم بضغطة واحدة إلى مستودع المشروع على GitHub.

---

## 📐 نظام توزيع التقديرات والمعدل (Grading Scale)


| النسبة المئوية | التقدير العام | المعدل التراكمي (GPA) |
| :--- | :--- | :--- |
| 90% وأعلى | امتياز 🌟 | 4.0 |
| 85% إلى 89% | جيد جداً مرتفع ✅ | 3.7 |
| 80% إلى 84% | جيد جداً ✅ | 3.3 |
| 75% إلى 79% | جيد مرتفع 🆗 | 3.0 |
| 65% إلى 74% | جيد 🆗 | 2.4 |
| 50% إلى 64% | مقبول ⚠️ | 1.0 إلى 2.0 |
| أقل من 50% | راسب ❌ | 0.0 |

---

## 🛠️ التكنولوجيا المستخدمة (Built With)

*   **Language:** Python 3.x
*   **Library:** Tkinter (واجهة المستخدم الرسومية)
*   **Library:** Webbrowser (لربط الأزرار بالإنترنت)

---

## 💻 طريقة التشغيل (How to Run)

1. تأكد من تثبيت لغة **Python** على جهازك.
2. قم بتحميل ملف الكود `grades.py`.
3. افتح موجه الأوامر (Terminal) واكتب الأمر التالي لتشغيل التطبيق:
   ```bash
   python grades.py
   ```
