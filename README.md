🎓 Student Grade & GPA Calculator
برنامج بسيط بواجهة رسومية لحساب درجات الطلاب والمعدل التراكمي.
🚀 المميزات:
• حساب المجموع الكلي من 500.
• حساب النسبة المئوية المئوية بدقة.
• تحويل النسبة إلى معدل تراكمي (GPA) من 4.0.
• تحديد التقدير العام (امتياز، جيد جداً، إلخ).
• واجهة مستخدم حديثة وسهلة باستخدام مكتبة Tkinter.
🛠️ التكنولوجيا المستخدمة:
• Python.
• Tkinter Library (لواجهة المستخدم).
